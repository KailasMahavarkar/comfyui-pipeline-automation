Purpose: Core module for Save As node (Phase 2). Every generated image gets metadata embedded for traceability.
Output: lib/metadata.py with per-format handlers and a read-back function, plus comprehensive tests.

1. Create `lib/metadata.py` implementing:

   Public API:
   - `embed_metadata(filepath: Path, tags: list[str], prompt: str, format_type: str) -> None`
     - Builds flat text: f"prompt: {prompt}\ntags: {', '.join(tags)}"
     - Dispatches to format-specific handler via dict lookup
     - Raises ValueError for unknown formats
   - `read_metadata(filepath: Path, format_type: str) -> Optional[str]`
     - Reads back embedded metadata string
     - Returns None if no metadata found

   Internal handlers:
   - `_embed_png(filepath, text)`: Open with Pillow, create PngInfo, add_text("comfyui_metadata", text), save with pnginfo=
   - `_embed_jpeg(filepath, text)`: Open with Pillow, get existing EXIF via img.info.get("exif", b""). If empty, create fresh exif_dict: {"0th": {}, "Exif": {}, "GPS": {}, "1st": {}}. Set ImageDescription = text.encode("utf-8"). Use piexif.dump() then piexif.insert() to write back (NOT img.save with exif= which is lossy).
   - `_embed_webp(filepath, text)`: Open with Pillow, build XMP bytes with proper xpacket wrapper and dc:description. Save with xmp= parameter. XMP MUST be bytes.
   - `_embed_gif(filepath, text)`: Open with Pillow, save with comment=text.encode("utf-8") parameter.

   Read-back handlers:
   - `_read_png(filepath)`: Open, return img.text.get("comfyui_metadata")
   - `_read_jpeg(filepath)`: Open, load EXIF, return ImageDescription decoded
   - `_read_webp(filepath)`: Open, parse XMP from img.info.get("xmp"), extract dc:description content
   - `_read_gif(filepath)`: Open, return img.info.get("comment", b"").decode("utf-8")

   CRITICAL pitfall handling per research:
   - JPEG: Use piexif.insert() on the file directly instead of re-saving through Pillow (avoids quality loss)
   - PNG: Always create fresh PngInfo (Pillow does NOT preserve existing PngInfo on save)
   - WebP: Always encode XMP to bytes before passing to save()
   - All: Use pathlib.Path, no os.path

2. Run all tests. Fix any failures. Refactor for clarity.

3. Update `lib/__init__.py` to export: `from .metadata import embed_metadata, read_metadata`
